搭建博客的过程（跟敲代码）

具体步骤

node.js下载安装，因为hexo基于node生成

terminal
-sudo su //切换至root用户
npm install -g cnpm –registry=https://registry.npm.taobao.org
淘宝镜像源 更快一些
安装hexo博客框架
cnpm install -g hexo-cli //全局安装hexo
mkdir blog //当前路径下创建新目录
cd blog //进入blog目录
pwd //查看当前路径
sudo hexo init //初始化
hexo s //hexo strat/server http://localhost:4000
hexo n “搭建博客的过程（跟敲代码）”. (这里的引号得是英文）
cd sources/_posts/ //进入创建md文件的路径
vim 搭建博客的过程（跟敲代码）.md //进入md文件开始写

vim状态下，
esc + i 进入insert 模式，
markdown 语句

当我们按ESC键后，接着再输入:号时，vi会在屏幕的最下方等待我们输入命令；
:w 保存；
:w filename 另存为filename；
:wq! 保存退出；
:wq! filename 注：以filename为文件名保存后退出；
:q! 不保存退出；
:x 应该是保存并退出 ，功能和:wq!相同
